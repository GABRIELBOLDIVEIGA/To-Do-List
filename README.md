## Scripts disponíveis

Nesse projeto você pode rodar o comando:

### `npm start`

Executa o aplicativo no modo de desenvolvimento.\
No navegador padrão a porta: [http://localhost:3000](http://localhost:3000) estará rodando a aplicação.
